# 📊 Finance Analyst Dashboard | Power BI

An interactive **Finance Analyst Dashboard** built using **Microsoft Power BI** to analyze financial performance and support data-driven decision-making. The dashboard provides meaningful insights through dynamic visualizations, KPIs, and interactive filters.

## 🚀 Features
- 📈 Revenue, Profit & Expense KPI Cards
- 📅 Monthly & Yearly Financial Trend Analysis
- 🌍 Region-wise Performance Analysis
- 📦 Category & Product Performance
- 👥 Customer Performance Analysis
- 🎯 Dynamic Filters & Slicers
- 🔍 Drill-through & Cross-filtering
- 📊 Interactive Charts & Visualizations

## 🛠️ Tools & Technologies
- 📊 Microsoft Power BI
- ⚡ Power Query
- 🧮 DAX (Data Analysis Expressions)
- 🗂️ Data Modeling
- 📑 Microsoft Excel

## 💡 Key Insights
- 📈 Identified revenue and profit trends over time.
- 🏆 Highlighted top-performing regions and categories.
- 💰 Compared financial performance across multiple dimensions.
- 📊 Enabled interactive analysis using slicers and filters.
- 📌 Supported better business decision-making through data visualization.
